# Lead CRM

Мини-CRM система для распределния лидов между операторами по источникам.

# Функциональность

- Управление операторами и их нагрузкой
- Настройка источников (ботов)
- Распределение обращений с учётом весов операторов
- Автоматическое назначение операторов с учётом лимитов нагрузки

## Технологии

- **FastAPI** - веб-фреймворк
- **SQLAlchemy** - ORM
- **SQLite** - база данных
- **Alembic** - миграции базы данных
- **Pydantic** - валидация данных
